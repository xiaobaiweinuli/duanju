# 短剧机器人插件
微信短剧搜索机器人
## 支持功能
- 自动同意好友（同意后自动拉群）
- 个人免费/付费搜索(支持模糊搜索，支持下一页上一页)
- 自定义个人资源
- 入群自动欢迎语
- 免费/付费自动拉群
- 自定义回复内容（成功回复、失败回复、未付费回复），支持推广尾巴
- 在线后台管理界面（在线上传个人资源）
